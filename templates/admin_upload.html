Studio Bulk Registration

Upload a CSV file prepared by a studio to register multiple students at once. No payment is charged — registrations are marked Invoiced and you bill the studio separately.

How It Works

1
Share the CSV template with the studio (or send them the download link from the registration page).
2
The studio fills in one row per student and emails the completed file back to you.
3
Upload the file here. Students are imported instantly and appear in the dashboard.
4
Invoice the studio separately for the total amount owed.

CSV Columns

studio_name
Studio's full name
studio_email
Studio email — receives QR if no student email
first_name
Student first name (required)
last_name
Student last name (required)
gender
Female / Male / Non-binary
birth_date
MM/DD/YYYY
email
Contact email
phone
Phone number
mobile
Mobile number (optional)
is_title
yes / no — title registrant?
routine_name
Routine name (title only)
reg_type
workshop / opening / both
tshirt_size
